# 主射咀大类数据字典 — 前端对接

数据来自《主射咀大类》Excel/截图（主射咀本体含加热/不加热/转接环、SR 球头、加热器、感温线、本体材质等），存于 **`sel_dict_category` / `sel_dict_item`**，分类 `code` 以 **`hrspec_mnz_`** 开头，与扁平行热流道规格中的主射咀字段（`hrspec_main_nozzle_*`，如是否加热、EPM 加热器等）**独立**；业务上可并行选用或后续再挂表字段。

## 拉取全部主射咀大类选项包

```http
GET /api/v1/selection-catalog/dict/main-nozzle-detail-options
Authorization: Bearer <token>
```

权限：`selection:read`。

响应与模具/热流道规格字典包相同：`{ "categories": { "<code>": [ { "id", "label", "sort_order" } ] } }`。

前端封装：`import { fetchMainNozzleDetailDictBundle } from "@/lib/selectionCatalogDictApi"`。

## 分类 `code` 一览（与 `backend/app/constants/sel_main_nozzle_detail_dict.py` 一致）

| `code` | 含义（截图区块） |
|--------|------------------|
| `hrspec_mnz_body_heated` | 主射咀本体 — 加热型 |
| `hrspec_mnz_body_unheated` | 主射咀本体 — 不加热型 |
| `hrspec_mnz_adapter_ring_bridge` | 其他配件 — 主射咀转接环（搭桥） |
| `hrspec_mnz_adapter_ring_stack` | 其他配件 — 主射咀转接环（叠模），截图无选项时可空 |
| `hrspec_mnz_sr_ball` | SR 球头 |
| `hrspec_mnz_main_heater` | 主射咀加热器（铜套/弹簧/镶嵌丝等） |
| `hrspec_mnz_thermocouple_style` | 感温线样式 |
| `hrspec_mnz_body_material` | 主射咀本体材质 |

**说明**

- 热流道规格表中的 **`hrspec_main_nozzle_heating` / `hrspec_main_nozzle_body_material` / `hrspec_main_nozzle_heater`**（是/否、材质简表、EPM 等）仍在 **`GET …/dict/hot-runner-spec-options`**，本包不替代；若需与扁平行字段统一 UUID，需在领域模型中单独设计映射。
- 感温线样式中「Φ1.0 镶嵌」下两条线长已拆成独立选项文案（如 `Φ1.0镶嵌 L100+1000`）。
- 按分类维护与全局字典相同：`GET/POST/PATCH/DELETE …/dict/categories`、`…/dict/items?category_code=…`（写操作需 `selection:write`）。管理端页面：**主射咀大类字典**（`/selection-catalog/main-nozzle-dict`）。
