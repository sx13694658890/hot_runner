# 模具选型数据模型文档

## 概述

本文档定义了模具选型系统的完整数据模型，基于以下四类参数：

1. **模具信息** - 模具基础配置和选型选项
2. **产品信息** - 产品特性和需求参数
3. **材料属性** - 43种塑料材料的成型工艺参数
4. **热流道系统** - 热流道组件配置选项

---

## 五、关联规则

### 5.1 材料约束规则

| 规则 | 触发条件 | 推荐配置 | 原因 |
|------|---------|---------|------|
| 高温材料规则 | 材料为 PEEK/PEI/PES/PPS等 | FS136材质、K型感温线、MCM加热器 | 高温材料需要耐高温组件 |
| 腐蚀性材料规则 | 材料为 PVC/POM等 | FS136材质、Ni/DLC镀层 | 腐蚀性材料会腐蚀普通钢材 |
| 高吸湿材料规则 | 材料为 PA6/PA66等 | 需预干燥、限制存胶时间 | 高吸湿材料需充分干燥 |
| 低粘度材料规则 | 材料为 PP/PE等 | 推荐针阀式热流道 | 低粘度材料易流涎 |
| 高粘度材料规则 | 材料为 PC/PEEK等 | 大流道直径、高注射压力 | 高粘度材料流动困难 |
| 透明材料规则 | 材料为 PMMA/PC等 | 铍铜咀芯、严格平衡 | 透明材料对外观要求高 |

### 5.2 产品需求规则

| 规则 | 触发条件 | 推荐配置 | 原因 |
|------|---------|---------|------|
| 高亮外观规则 | 外观要求 = 高亮 | 开放式热咀(OA,PLT,SOE) | 避免浇口痕迹 |
| 精密件规则 | 精度 = 精密件 | 针阀式热流道、严格平衡 | 需要精确控制注射 |
| 薄壁件规则 | 壁厚 < 1mm | 高注射压力、铍铜咀芯 | 需要快速充填 |
| 厚壁件规则 | 壁厚 > 3mm | 大胶口、冰水冷却 | 需要充分冷却 |
| 大批量规则 | 批量 ≥ 10万模次 | FS136材质、DLC镀层 | 需要高耐磨材质 |
| 换色规则 | 颜色 = 换色 | 针阀式热流道 | 便于清洗 |

### 5.3 热流道约束规则

| 规则 | 触发条件 | 强制要求 |
|------|---------|---------|
| 针阀必须有驱动 | 热流道类型 = 针阀式 | 必须配置驱动器(气动/油压) |
| 双色模必须有分流板 | 模具类型 = 双色 | 必须使用分流板系统 |
| 换色必须用针阀 | 颜色 = 换色 | 必须使用针阀式热流道 |

## 六、选型决策示例

### 示例1：汽车保险杠

**输入需求：**
- 应用领域：汽车
- 外观要求：晒纹

**推荐结果：**
- 材料：PP, ABS, PA6, PA66, PC+ABS
- 热流道类型：含分流板的开放式 / 含分流板的针阀式 / 单咀

- 主射咀材质：FS136
- 咀芯材质：钨铜, 钼合金

- 外观要求：高亮
- 颜色：透明
- 尺寸精度：精密件
- 生产批量：中批量

**推荐结果：**
- 材料：PP, PE, PC, PMMA, POM
