# 热咀大类数据字典 — 前端对接

数据来自《热咀》Excel/截图（本体、截面、咀头、衬套、运水套等），存于 **`sel_dict_category` / `sel_dict_item`**，分类 `code` 以 **`hrspec_hnz_`** 开头，与扁平行热流道规格中的 **`hrspec_hot_nozzle_*`、浇口直径、咀芯等** 字段独立；结构代码在库内为**扁平选项**，开放式/针阀式/大水口/点浇口的级联可在前端按产品规则实现。

## 拉取全部热咀大类选项包

```http
GET /api/v1/selection-catalog/dict/hot-nozzle-detail-options
Authorization: Bearer <token>
```

权限：`selection:read`。

响应：`{ "categories": { "<code>": [ { "id", "label", "sort_order" } ] } }`。

前端封装：`import { fetchHotNozzleDetailDictBundle } from "@/lib/selectionCatalogDictApi"`。

## 分类 `code` 一览（与 `backend/app/constants/sel_hot_nozzle_detail_dict.py` 一致）

| `code` | 含义（截图区块） |
|--------|------------------|
| `hrspec_hnz_body_base_type` | 热咀本体 — 底座类型 |
| `hrspec_hnz_heaters_per_body` | 热咀本体 — 单个本体加热器数 |
| `hrspec_hnz_body_heater` | 热咀本体 — 热咀加热器 |
| `hrspec_hnz_body_material` | 热咀本体 — 热咀本体材质 |
| `hrspec_hnz_section_coiled` | 截面 — 绕线咀 |
| `hrspec_hnz_section_beryllium` | 截面 — 铍铜咀 |
| `hrspec_hnz_body_length` | 本体长度 |
| `hrspec_hnz_structure_code` | 热咀咀头 — 热咀结构代码（全量码表，扁平） |
| `hrspec_hnz_gate_diameter` | 热咀咀头 — 胶口直径 |
| `hrspec_hnz_core_material` | 热咀咀头 — 咀芯材质 |
| `hrspec_hnz_core_coating` | 热咀咀头 — 咀芯涂层工艺 |
| `hrspec_hnz_nozzle_cap` | 热咀咀头 — 咀帽 |
| `hrspec_hnz_insulation_ring` | 热咀咀头 — 隔热圈 |
| `hrspec_hnz_valve_bushing` | 热咀咀头 — 阀口套 |
| `hrspec_hnz_outer_circlip` | 热咀咀头 — 外卡簧 |
| `hrspec_hnz_bushing` | 衬套（截图无选项时可空） |
| `hrspec_hnz_water_jacket` | 运水套 |

**说明**

- 热流道规格表 **`GET …/dict/hot-runner-spec-options`** 中的热咀结构、浇口、咀芯等仍为扁平行业务字段；本包为**热咀大类扩展**，按需与规格表并行或后续挂表。
- 按分类维护：`GET/POST/PATCH/DELETE …/dict/items?category_code=…`（写操作需 `selection:write`）。管理端：**热咀大类字典**（`/selection-catalog/hot-nozzle-dict`）。
