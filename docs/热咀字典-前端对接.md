# 热咀大类数据字典 — 前端对接

数据来自《热咀》Excel/截图（本体、截面、咀头、衬套、运水套等），存于 **`sel_dict_category` / `sel_dict_item`**，分类 `code` 以 **`hrspec_hnz_`** 开头，与扁平行热流道规格中的 **`hrspec_hot_nozzle_*`、浇口直径、咀芯等** 字段独立；咀头区含结构代号四类及胶口套、咀芯、咀帽、阀口套、外卡簧、隔热帽等，选项为各分类下扁平码表。

## 拉取全部热咀大类选项包

```http
GET /api/v1/selection-catalog/dict/hot-nozzle-detail-options
Authorization: Bearer <token>
```

权限：`selection:read`。

响应：`{ "categories": { "<code>": [ { "id", "label", "sort_order" } ] } }`。

前端封装：`import { fetchHotNozzleDetailDictBundle } from "@/lib/selectionCatalogDictApi"`。

## 分类 `code` 一览（与 `backend/app/constants/sel_hot_nozzle_detail_dict.py` 一致）

| `code` | 含义（截图区块） |
|--------|------------------|
| `hrspec_hnz_body_base_type` | 热咀本体 — 底座类型 |
| `hrspec_hnz_heaters_per_body` | 热咀本体 — 单个本体加热器数 |
| `hrspec_hnz_body_heater` | 热咀本体 — 热咀加热器 |
| `hrspec_hnz_body_material` | 热咀本体 — 热咀本体材质 |
| `hrspec_hnz_section_coiled` | 垫圈 — 镶嵌用 |
| `hrspec_hnz_section_beryllium` | 垫圈 — 铜套用 |
| `hrspec_hnz_body_length` | 本体碟簧 |
| `hrspec_hnz_structure_open_large` | 热咀咀头 — 开放式-大水口（TOE、SOE、TLS、OA、PLT、PLS、TLC、SLC） |
| `hrspec_hnz_structure_open_dot` | 热咀咀头 — 开放式-点胶口（CS、TAC、SAC、SCC、CC、CCH、CA、CT、CTC） |
| `hrspec_hnz_structure_valve_large` | 热咀咀头 — 针阀式-大水口（STVL、TVL） |
| `hrspec_hnz_structure_valve_dot` | 热咀咀头 — 针阀式-点胶口（EVV、SVV、VV、TVA、TVAP、VA、STVA） |
| `hrspec_hnz_gate_diameter` | 热咀咀头 — 胶口套（Φ 规格选项，code 仍为 `gate_diameter`） |
| `hrspec_hnz_core_material` | 热咀咀头 — 咀芯材质 |
| `hrspec_hnz_core_coating` | 热咀咀头 — 咀芯涂层工艺 |
| `hrspec_hnz_nozzle_cap` | 热咀咀头 — 咀帽 |
| `hrspec_hnz_valve_bushing` | 热咀咀头 — 阀口套 |
| `hrspec_hnz_outer_circlip` | 热咀咀头 — 外卡簧 |
| `hrspec_hnz_insulation_ring` | 热咀咀头 — 隔热帽 |
| `hrspec_hnz_bushing` | 衬套（截图无选项时可空） |
| `hrspec_hnz_water_jacket` | 运水套 |

**说明**

- 热流道规格表 **`GET …/dict/hot-runner-spec-options`** 中的热咀结构、浇口、咀芯等仍为扁平行业务字段；本包为**热咀大类扩展**，按需与规格表并行或后续挂表。结构代码已按型式拆为上述四个 `hrspec_hnz_structure_*` 分类（已废弃扁平 `hrspec_hnz_structure_code`）。
- 按分类维护：`GET/POST/PATCH/DELETE …/dict/items?category_code=…`（写操作需 `selection:write`）。管理端：**热咀大类字典**（`/selection-catalog/hot-nozzle-dict`）。
