# 驱动系统数据字典 — 前端对接

数据来自《驱动系统》Excel/截图（阀针、阀套、气缸板开孔驱动器、分流板固定驱动器、单点针阀补充零件、导向套等），存于 **`sel_dict_category` / `sel_dict_item`**，分类 `code` 以 **`hrspec_drv_`** 开头，与扁平行热流道规格中的 **阀针样式/材质/镀层（`hrspec_valve_*`）** 独立；型号与 BOM 拆成多条分类，级联与依赖关系由前端或业务规则组合。

## 拉取全部驱动系统选项包

```http
GET /api/v1/selection-catalog/dict/drive-system-detail-options
Authorization: Bearer <token>
```

权限：`selection:read`。

响应：`{ "categories": { "<code>": [ { "id", "label", "sort_order" } ] } }`。

前端封装：`import { fetchDriveSystemDetailDictBundle } from "@/lib/selectionCatalogDictApi"`。

## 分类 `code` 一览（与 `backend/app/constants/sel_drive_system_detail_dict.py` 一致）

| `code` | 含义（截图区块） |
|--------|------------------|
| `hrspec_drv_valve_pin_spec` | 阀针 — 阀针规格 |
| `hrspec_drv_valve_pin_material` | 阀针 — 阀针材质 |
| `hrspec_drv_valve_pin_coating` | 阀针 — 阀针镀层工艺 |
| `hrspec_drv_sleeve_regular` | 阀套 — 常规阀套（直径） |
| `hrspec_drv_sleeve_color_change` | 阀套 — 换色阀套（直径） |
| `hrspec_drv_actuator_plate_model` | 在气缸板上开孔的驱动器 — 型号 |
| `hrspec_drv_actuator_plate_bom_hs_vc` | 气缸板开孔 — HS/VC 系共用零件 |
| `hrspec_drv_actuator_plate_bom_fep` | 气缸板开孔 — FEP30 零件 |
| `hrspec_drv_manifold_pneumatic_connector` | 固定在分流板上 — 整体式气缸 — 接头角度 |
| `hrspec_drv_manifold_hydraulic_connector` | 固定在分流板上 — 整体式油缸 — 接头角度 |
| `hrspec_drv_manifold_tubing` | 固定在分流板上 — 管材类型（铁氟龙管/铁管） |
| `hrspec_drv_single_point_base_parts` | 单点针阀补充零件 — 底座组成 |
| `hrspec_drv_guide_sleeve` | 导向套 — 规格 |

**说明**

- 热流道规格表 **`GET …/dict/hot-runner-spec-options`** 中阀针相关字段仍为扁平行选型；本包为**驱动系统扩展**。
- 按分类维护：`GET/POST/PATCH/DELETE …/dict/items?category_code=…`（写操作需 `selection:write`）。管理端：**驱动系统字典**（`/selection-catalog/drive-system-dict`）。
